# ジャグラー データ集計ツール

ジャグラーシリーズのホールデータを手軽に集計できるWebアプリです。  
スマートフォンで素早く入力できる操作性を重視しています。

---

## このツールでできること

- 台ごとに **通常G・BB回数・RB回数** を入力
- 全台の **BB確率・RB確率・合算確率** を自動計算・表示
- **通常Gを直接入力** する方法と、**合成確率から逆算** する方法の両方に対応

---

## 通常Gの入力方法

### 方法①：通常Gを直接入力

通常ゲーム数がわかっている場合は、「通常G」欄にそのまま入力します。

### 方法②：合成確率から通常Gを逆算

通常ゲーム数がわからず合成確率しかわからない場合は、  
「合成」欄に **合成確率の分母** を入力します。

**例：合成確率が 1/120 → 「120」と入力**

通常Gは以下の式で自動計算されます：

```
通常G = 合成確率の分母 × (BB回数 + RB回数)
```

例）合成: 120、BB: 10回、RB: 8回 → 通常G = 120 × 18 = **2160G**

自動計算された通常Gは、通常G欄に **青い斜体で「2160G 自動」** と表示されます。

---

## 計算式

| 項目     | 計算式                                |
|----------|---------------------------------------|
| BB確率   | 通常G合計 ÷ BB合計                    |
| RB確率   | 通常G合計 ÷ RB合計                    |
| 合算確率 | 通常G合計 ÷ (BB合計 + RB合計)         |

---

## スマホでの使い方

1. アプリを開き、パスワードを入力してログイン（初期値：`123456`）
2. 台番号・通常G（または合成）・BB・RBを入力
3. 数字キーボードの **「.（ドット）」** を押すと次の欄へ自動移動
4. 画面下部に確率が自動表示されます

### 入力の流れ

```
台番 → 通常G → 合成 → BB → RB → （次の行の）台番 → …
```

RB欄で「.」を押すと次の行へ移動。最終行のRBで「.」を押すと行が自動追加されます。

---

## パスワードの変更方法

`script.js` をテキストエディタで開き、先頭の以下の部分を変更してください：

```javascript
const PASSWORD = "123456"; // ← ここを6桁の好きな数字に変更
```

変更後、ファイルを保存してGitHubにアップロードし直してください。

---

## ローカルで開く方法

1. `index.html`・`style.css`・`script.js`・`README.md` を同じフォルダに置く
2. `index.html` をブラウザにドラッグ＆ドロップ（またはダブルクリック）して開く

---

## GitHub Pages で公開する方法

1. GitHubにアカウントを作成・ログイン
2. 新しいリポジトリを作成（名前は何でもOK、例：`juggler-app`）
3. 4つのファイルをすべてアップロード
4. リポジトリの **Settings → Pages** を開く
5. **Source** を `Deploy from a branch` → ブランチを `main`、フォルダを `/(root)` に設定
6. **Save** を押す
7. 数分後に以下のURLでアクセスできます：

```
https://（あなたのGitHubユーザー名）.github.io/（リポジトリ名）/
```

---

## データのリセット方法

- アプリ内の **「全削除」** ボタンを押す（ログイン状態は維持されます）
- または、ブラウザの設定からサイトデータ（localStorage）を削除する

---

## 注意事項

このアプリのパスワード機能は**簡易的なアクセス制限**です。  
URLを知られると、ブラウザの開発者ツールからデータへアクセスできる可能性があります。  
個人・身内での使用を想定した機能であり、機密情報の保護には使用しないでください。
